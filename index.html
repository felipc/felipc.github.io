Acessibilidade na Web e técnicas para aumentar a acessibilidade em RIA (Rich Web Applications) para usuários com dificuldades visuais

Projeto de pesquisa de Mestrado

Este projeto de pesquisa visa fazer um estudo bibliográfico sobre o tema de Acessibilidade na Web, especificamente sobre a acessibilidade para usuários com dificuldades visuais; junto com uma análise de técnicas existentes na literatura propostas para ampliar a acessibilidade na Web e uma proposta de alguma melhoria ou técnica nova sobre a acessibilidade em RIA (Rich Web Applications)

Abstract

The initial part of this project is to perform a literature review on web accessibility, more specifically on the accessibility aspects for visually-impaired users, in order to obtain an understanding of the existing methods and techniques currently proposed on the literature. With this understanding, the project will focus on the current work on automatically augmenting accessibility on Rich Web Applications for visually-impaired users, and to propose some improvement over an existing technique, or an additional new technique, that can improve the experience for these users when using the Web.

Contexto

A web de hoje é muito diferente da web em sua forma inicial. Antes um conjunto de textos, conteúdo estático, imagens e hiperlinks, a web de hoje em dia possui uma característica predominantemente dinâmica. Em grande parte, o conteúdo dos websites são carregados dinâmicamente, de acordo com ações dos usuários, e não podem ser previstos antecipadamente no momento de geração da página. Isso possibilita a criação de aplicações que rodam na web, como clientes de e-mail, sistemas de mapas e navegação, aplicativos de comunicação com mensagens instantâneas, etc. Entretanto, não só aplicações fazem uso destas técnicas, que também são usadas em sites como jornais, blogs, albúm de fotografias, vídeos, entre outros, que eram históricamente sites estáticos, porém apresentam também características de aplicações web hoje em dia.

Esta dinamicidade, apesar de proporcionar toda uma nova gama de interações possíveis na web e tornar a sua experiência mais rica e interativa, causa diversos problemas para usuários com todos tipos de necessidades especiais (motoras, visuais, cognitivas, etc.). Estes problemas partem tanto da sobrecarga de informações imposta por tais características, bem como das limitações das ferramentas de acessibilidade existentes, que muitas vezes não conseguem lidar bem com o fluxo de conteúdo não-estático. Além disso, as necessidades especiais de usuários são muitas vezes esquecidas ou ignoradas pelos desenvolvedores web, que não testam ou não estruturam suas páginas de forma à prover todos os atributos e estrutura necessários para as ferramentas de acessbilidade lidarem com suas aplicações.

Desta forma, existe uma necessidade de um melhor entendimento dos métodos envolvidos na criação de conteúdo dinâmico propriamente acessíveis, bem como o uso de técnicas para cobrir as lacunas deixadas (propositalmente ou não) pelos desenvolvedores das aplicações web existentes.

Objetivos

O objetivo principal deste trabalho é estudar as técnicas existentes ao redor da acessibilidade na web para usuários de deficiências visuais, bem como obter um entendimento de como tais técnicas podem ser propostas e utilizadas de forma automática para melhorar a experiência destes usuários com aplicações web.

Um protótipo será criado de forma a analisar e mensurar a qualidade de diversas técnicas, de acordo com especificações e benchmarks já existentes na literatura de acessibilidade computacional. Espera-se conseguir propor alguma modificação ou técnica nova que consiga melhorar os resultados de acessibilidade de acordo com algum destes benchmarks.
